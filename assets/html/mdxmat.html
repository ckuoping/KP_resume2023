Material Wizard

Polymer and non-polymer material database

User Research

Collect user feedback on polymer material retrieval behavior, identify weaknesses in the previous version, and define future development specifications.

Web Development

Using Adobe XD to complete UIUX mockups, and using Angular and front-end languages ( HTML, CSS, TypeScript, and Chart.js) to develope web application.

Desktop Application

The web application was packaged into a desktop application using Electron and included in the Moldex3D Studio installation package.

Key Features

Material Data and Curves

Material Wizard provides information of thousands of materials. The material list is displayed using a tree hierarchy component and material properties are horizontally arranged using tab components. Material data are visualized as line charts using Chart.js.


Create and Modify Material

Material Wizard also provides a material data editing feature. Using the Stepper component, properties of a material are listed, and parameters of each property are displayed using the table component. After the user completes all steps, the material editing is done.
